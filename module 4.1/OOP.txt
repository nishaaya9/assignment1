Q-2. What is the difference between OOP and POP?
Ans-
                               OOP                                                                                                             POP

1. OOP stands for Object-Oriented Programming.                      1. POP stands for Procedural Oriented Programming.

2. OOP follows a bottom-up approach.                                          2. POP follows a top-down approach.

3. A program is divided into objects and their interactions.      3. A program is divided into functions and they interact.

4. Inheritance is supported.                                                              4. Inheritance is not supported.

5. Access control is supported via access modifiers.                5. No access modifiers are supported.

6. Encapsulation is used to hide data.                                            6. No data hiding present. Data is globally accessible.

7. Ex.  C++, Java                                                                                  7. Ex.   C, Pascal
